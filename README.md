# NYC Parks App Backend

Express and Nodejs backend for the NYC Parks app.

The NYC Parks app is Elijah Morton's January 2026 Winter Term project for Oberlin College.

## Installation
 - Set up a MySQL database for the backend to interact with.
 - Run `npm install` from the project's root directory.
 - Create a `.env` file in the home directory based on `.env.example` and fill it in.

## Running
 - Start with `npm run dev`
